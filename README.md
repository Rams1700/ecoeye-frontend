EcoEye is a full-stack web application designed to promote environmental awareness by allowing users to report garbage issues, purchase eco-friendly products, and track environmental impact.

## Live Demo
Frontend: https://ecoeye-frontend.vercel.app  
Backend API: https://ecoeye-backend.onrender.com  

##  Tech Stack
- Frontend: HTML, CSS, JavaScript (Vercel)
- Backend: Node.js, Express.js (Render)
- Database: MongoDB Atlas

## Features
- Report garbage with live location
- Contact form connected to backend
- Shopping cart & order placement
- MongoDB data storage
- Fully deployed full-stack project

## Deployment
- Frontend deployed on **Vercel**
- Backend deployed on **Render**
- Database hosted on **MongoDB Atlas**
